# 🚀 Быстрый запуск

## За 5 минут до работающего бота!

### 1. Получите токен бота (2 минуты)
1. Откройте Telegram
2. Найдите **@BotFather**
3. Отправьте `/newbot`
4. Придумайте имя: `Хоккейная Академия`
5. Придумайте username: `your_hockey_bot`
6. **Скопируйте токен!**

### 2. Настройте бота (1 минута)
```bash
# Установите библиотеки
pip install python-telegram-bot python-dotenv

# Скопируйте пример конфигурации
cp .env.example .env
```

**Откройте `.env` и вставьте ваш токен:**
```env
BOT_TOKEN=ВАШ_ТОКЕН_ЗДЕСЬ
```

### 3. Запустите бота (1 минута)
```bash
python hockey_bot_updated.py
```

### 4. Тестируйте! (1 минута)
1. Найдите вашего бота в Telegram
2. Отправьте `/start`
3. Заполните профиль
4. Попробуйте записаться на тренировку!

## 🔧 Стать администратором

1. Узнайте ваш ID: отправьте сообщение боту **@userinfobot**
2. Добавьте ID в `config.py`:
```python
ADMIN_IDS = [ВАШ_ID_ЗДЕСЬ]
```
3. Перезапустите бота
4. Теперь у вас есть кнопка "🔧 Админ-панель"

## ⚡ Быстрые команды

**Добавить тренировку:**
```
/add_training Новая тренировка|Описание|2024-12-25 19:00|Тренер|15|Средний|2000
```

**Отправить всем сообщение:**
```
/broadcast Привет! Новая тренировка завтра!
```
