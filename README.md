# 🏒 Galaxy Hockey Academy Telegram Bot

Крутой Telegram бот для хоккейной академии Galaxy Hockey Academy в Дубае.

## 🌟 Возможности бота

- **📅 Расписание тренировок** - Полное расписание на всю неделю
- **📝 Регистрация на тренировки** - Простая форма записи с сохранением данных
- **🏒 Программы обучения** - 4 различных программы для всех уровней
- **📞 Контактная информация** - Адрес, телефон, часы работы
- **📰 Новости академии** - Последние новости и события
- **💡 Советы по хоккею** - Полезные советы от профессиональных тренеров
- **🤖 Умные ответы** - Автоматические ответы на частые вопросы

## 🚀 Быстрый запуск

### Вариант 1: Автоматическая установка (Linux/Mac)

```bash
chmod +x start_bot.sh
./start_bot.sh
```

### Вариант 2: Ручная установка

1. **Установите зависимости:**
```bash
pip3 install -r requirements.txt
```

2. **Запустите бота:**
```bash
python3 hockey_bot.py
```

## 📋 Требования

- Python 3.8 или выше
- pip (устанавливается с Python)
- Интернет соединение

## 🛠 Настройка

### Смена токена бота

Если вы хотите использовать другой токен, отредактируйте файл `hockey_bot.py`:

```python
BOT_TOKEN = "ВАШ_НОВЫЙ_ТОКЕН_ЗДЕСЬ"
```

### Настройка расписания

Вы можете изменить расписание тренировок в классе `HockeyBot`:

```python
self.training_schedule = {
    "Monday": "16:00-17:30 - Beginners, 18:00-19:30 - Advanced",
    # Добавьте или измените расписание
}
```

### Изменение программ обучения

Программы обучения настраиваются в `self.training_programs`:

```python
self.training_programs = {
    "🥅 Beginner Program": {
        "description": "Описание программы",
        "duration": "8 weeks",
        "price": "1200 AED",
        "age": "6+ years"
    }
}
```

## 🎯 Команды бота

- `/start` - Главное меню
- `/cancel` - Отменить текущую операцию (во время регистрации)

## 📱 Функции

### Главное меню
- Интерактивные кнопки для навигации
- Красивый интерфейс с эмодзи

### Регистрация на тренировки
- Пошаговая форма сбора данных
- Автоматические рекомендации программ
- Сохранение данных пользователей

### Умные ответы
Бот автоматически отвечает на вопросы содержащие:
- Приветствия (привет, здравствуйте)
- Цены (цена, стоимость, сколько стоит)
- Время (время, расписание, когда)
- Местоположение (где, адрес, локация)

## 🗄 База данных

В текущей версии данные сохраняются в памяти. Для production использования рекомендуется:
- SQLite для небольших проектов
- PostgreSQL/MySQL для крупных проектов
- Redis для кэширования

## 🔧 Расширение функционала

### Добавление новых разделов

1. Создайте новую функцию:
```python
async def show_new_section(update: Update, context: ContextTypes.DEFAULT_TYPE) -> None:
    # Ваш код здесь
```

2. Добавьте кнопку в главное меню:
```python
keyboard = [
    [InlineKeyboardButton("🆕 Новый раздел", callback_data='new_section')]
]
```

3. Добавьте обработчик в `button_handler`:
```python
elif query.data == 'new_section':
    await show_new_section(update, context)
```

## 🐛 Решение проблем

### Ошибка "ModuleNotFoundError"
```bash
pip3 install --upgrade pip
pip3 install -r requirements.txt
```

### Ошибка "Unauthorized"
Проверьте токен бота в коде.

### Бот не отвечает
Убедитесь, что:
- Интернет подключение активно
- Токен бота правильный
- Бот добавлен в чат

## 📞 Поддержка

При возникновении проблем:
1. Проверьте логи в консоли
2. Убедитесь, что все зависимости установлены
3. Проверьте токен бота

## 🎨 Кастомизация

### Изменение текстов
Все тексты находятся в соответствующих функциях и легко редактируются.

### Добавление изображений
Используйте метод `send_photo()` для отправки изображений:
```python
await update.message.reply_photo(
    photo='URL_ИЗОБРАЖЕНИЯ',
    caption='Описание'
)
```

### Локализация
Создайте словари с переводами и переключайтесь между языками:
```python
TEXTS = {
    'ru': {'welcome': 'Добро пожаловать!'},
    'en': {'welcome': 'Welcome!'}
}
```

## 🚀 Деплой в продакшн

### Heroku
1. Создайте `Procfile`:
```
worker: python hockey_bot.py
```

2. Деплойте через Git

### VPS
1. Установите screen или tmux
2. Запустите бота в фоновом режиме
3. Настройте автозапуск через systemd

### Docker
```dockerfile
FROM python:3.9-slim
COPY . /app
WORKDIR /app
RUN pip install -r requirements.txt
CMD ["python", "hockey_bot.py"]
```
