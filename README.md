# 🏒 Telegram Бот Хоккейной Академии

Полнофункциональный Telegram бот для управления хоккейной академией с возможностью регистрации на тренировки, просмотра расписания и администрирования.

## 🌟 Возможности

### Для пользователей:
- 📝 **Регистрация профиля** с указанием возраста и уровня подготовки
- 🏒 **Запись на тренировки** с проверкой доступных мест
- 📅 **Просмотр расписания** всех тренировок
- 👨‍🏫 **Информация о тренерах** и их специализации
- 📱 **Управление записями** (отмена, просмотр)
- ℹ️ **Информация об академии** и контакты

### Для администраторов:
- 📊 **Статистика** пользователей и тренировок
- ➕ **Добавление новых тренировок**
- 📋 **Управление существующими тренировками**
- 👥 **Просмотр списка пользователей**
- 📨 **Массовая рассылка** сообщений

## 🚀 Установка и настройка

### 1. Подготовка

```bash
# Клонируйте проект или скопируйте файлы
git clone <your-repo-url>
cd hockey-bot

# Установите Python зависимости
pip install -r requirements.txt
```

### 2. Создание бота в Telegram

1. Откройте Telegram и найдите **@BotFather**
2. Отправьте команду `/newbot`
3. Введите название бота (например: "Хоккейная Академия")
4. Введите username бота (например: "hockey_academy_bot")
5. Скопируйте полученный токен

### 3. Настройка конфигурации

```bash
# Скопируйте файл конфигурации
cp .env.example .env

# Отредактируйте .env файл
nano .env
```

Заполните `.env` файл:

```env
# Токен от @BotFather
BOT_TOKEN=1234567890:ABCDEFGHIJKLMNOPQRSTUVWXYZ

# Настройки академии
ACADEMY_NAME=Хоккейная Академия "Ледяные Волки"
ACADEMY_ADDRESS=г. Москва, ул. Спортивная, 15
ACADEMY_PHONE=+7 (495) 123-45-67
ACADEMY_EMAIL=info@icewolves.ru
ACADEMY_WEBSITE=www.icewolves.ru

# База данных
DB_NAME=hockey_academy.db

# Уведомления
REMINDER_HOURS=24
```

### 4. Настройка администраторов

Отредактируйте файл `config.py` и добавьте ваш Telegram ID в список `ADMIN_IDS`:

```python
# Чтобы узнать ваш ID, отправьте любое сообщение боту @userinfobot
ADMIN_IDS = [
    123456789,  # Ваш Telegram ID
    987654321,  # ID других администраторов
]
```

### 5. Запуск бота

```bash
# Запуск основного бота
python hockey_bot_updated.py

# Или запуск старой версии (если нужна)
python hockey_bot.py
```

## 💾 База данных

Бот использует SQLite для хранения данных. База данных создается автоматически при первом запуске.

### Таблицы:
- **users** - информация о пользователях
- **trainings** - расписание тренировок
- **registrations** - записи пользователей на тренировки

## 📱 Использование бота

### Команды пользователя:
- `/start` - начать работу с ботом
- `/profile` - заполнить/обновить профиль
- `/schedule` - посмотреть расписание

### Команды администратора:
- `/admin` - открыть админ-панель
- `/add_training` - добавить тренировку
- `/broadcast` - отправить сообщение всем пользователям

### Формат добавления тренировки:
```
/add_training Название|Описание|ГГГГ-ММ-ДД ЧЧ:ММ|Тренер|Макс.участников|Уровень|Цена

Пример:
/add_training Техника броска|Отработка точности|2024-02-25 19:00|Иван Петров|12|Средний|2000
```

## 🔧 Расширение функционала

### Добавление новых тренеров
Отредактируйте функцию `coaches_info()` в файле `hockey_bot_updated.py`

### Изменение возрастных категорий
Отредактируйте функцию `get_skill_keyboard()` и соответствующую логику

### Добавление платежей
Интегрируйте Telegram Payments API или сторонние платежные системы

### Добавление уведомлений
Создайте cron job для отправки напоминаний о тренировках

## 🐛 Решение проблем

### Бот не отвечает
- Проверьте правильность токена в `.env`
- Убедитесь, что бот запущен
- Проверьте логи на наличие ошибок

### Ошибки базы данных
- Проверьте права доступа к файлу `hockey_academy.db`
- Удалите БД и перезапустите бота для пересоздания

### Админ-панель не работает
- Убедитесь, что ваш Telegram ID добавлен в `ADMIN_IDS`
- Перезапустите бота после изменения конфигурации
